## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **DO NOT** create a public issue.

Instead, please report it via [GitHub Security Advisories](https://github.com/carlos-camara/qa-hub-actions/security/advisories/new) or contact the maintainer directly.

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
